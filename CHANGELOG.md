## [1.0.0-beta.1] - 2026-01-17

### Added
- Core game engine with Canvas 2D rendering
- Turn-based artillery combat mechanics
- 11 unique weapons (Basic Shot, MIRV, Roller, Digger, Nuke, Mini Nuke, etc.)
- Destructible terrain with falling dirt physics
- AI opponents with Easy/Medium/Hard difficulty
- Economy system with shop between rounds
- 60 levels across 6 themed worlds
- Synthwave visual aesthetic with neon colors and CRT effects
- Responsive design supporting mobile and desktop
- iOS support via Capacitor
- TestFlight beta distribution setup
- Debug tools and TestAPI for development

### Technical
- Vanilla JavaScript ES6 modules (no build step required)
- Unified pointer API for mouse and touch input
- High-DPI support via devicePixelRatio scaling
- Web Audio API for music and sound effects
- Asset manifest system for swappable graphics
