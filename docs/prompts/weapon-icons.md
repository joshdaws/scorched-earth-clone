# Weapon Icon Prompts - 11 Types

Prompts for generating weapon icons for Scorched Earth: Synthwave Edition HUD weapon selection bar.

## Tool Recommendation

**Primary**: NanoBanana Pro
**Alternative**: Leonardo.ai (good transparency support)

## Style Consistency Reference

```
Primary colors:
- Neon Pink: #FF00FF
- Electric Cyan: #00FFFF
- Deep Purple: #8B00FF
- Hot Orange: #FF6600
- Neon Yellow: #FFFF00
- Radioactive Green: #00FF00

Common elements:
- Chrome/metallic surfaces
- Neon glow effects (2-4px blur)
- Clean silhouettes
- Geometric shapes
```

## Design Notes

Weapon icons must:
- **Consistent 64x64 pixels**: All icons same size for HUD bar
- **Clear silhouettes**: Instantly readable at small display sizes
- **Distinct shapes**: Each weapon visually unique from others
- **Synthwave style**: Neon glows, chrome metallic, bold colors
- **Transparent background**: For HUD overlay

### Icon Design Principles

1. **Simple geometry**: Icon should read clearly when small
2. **Bold outlines**: Help distinguish at a glance
3. **Color coding**: Weapons in related groups share accent colors
4. **Avoid text**: Rely on visual representation only

### Color Groupings

| Category | Weapons | Accent Color |
|----------|---------|--------------|
| Standard | Basic Shot, Missile, Big Shot | Cyan |
| Cluster | MIRV, Death's Head | Pink/Magenta |
| Roller | Roller, Heavy Roller | Orange |
| Digger | Digger, Heavy Digger | Purple |
| Nuclear | Mini Nuke, Nuke | Yellow/Green |

## Batch Generation Tips

1. **Generate by color group** for consistency:
   - Cyan group: Basic Shot, Missile, Big Shot
   - Pink group: MIRV, Death's Head
   - Orange group: Roller, Heavy Roller
   - Purple group: Digger, Heavy Digger
   - Nuclear group: Mini Nuke, Nuke

2. **Generate heavy variants alongside standard** to ensure visual hierarchy

3. **Test at actual HUD size** before finalizing (icons may appear smaller in-game)

4. **Generate 3-5 variations** per icon and select clearest silhouettes

5. **Maintain consistent glow intensity** across all icons

6. **Check color distinction** - each group should be immediately recognizable by color

## Post-Processing Notes

### Icon Uniformity
1. Ensure all icons have similar visual weight (none too light or heavy)
2. Center icons consistently within the 64x64 frame
3. Maintain similar glow intensities across the set

### Transparency Cleanup
1. No white fringing on edges
2. Glow should fade to full transparency
3. Test on both dark and light backgrounds

### HUD Integration Testing
1. Display all icons in a row to check visual coherence
2. Test at actual HUD scale (may be rendered smaller)
3. Ensure each icon is distinguishable at a glance
